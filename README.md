# bumbledb

An embedded, typed, set-semantic relational database. Currently at the drawing board:
five implementations (v1–v5) were built and discarded; this reset clears the repo to
rebuild the architecture docs first, decision by decision, before any code returns.

Prior implementations, their design docs, and the review that motivated this reset all
live in git history before the reset commit.

Contents:

- `docs/free-join-paper/` — Wang, Willsey, Suciu, *Free Join: Unifying Worst-Case
  Optimal and Traditional Joins* (arXiv:2301.10841v2), the algorithmic reference.
- `docs/architecture/` — the architecture docs, rewritten from scratch (in progress).
